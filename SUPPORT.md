# Support

## Getting Help

| Channel | Best for | Response time |
|---------|----------|---------------|
| [GitHub Discussions](https://github.com/totakit/totakit.github.io/discussions) | Questions, ideas, community help | Community-driven |
| [GitHub Issues](https://github.com/totakit/totakit.github.io/issues) | Bug reports, feature requests | 48 hours |
| [hello@totakit.com](mailto:hello@totakit.com) | Private inquiries, partnerships | 48 hours |
| [support@totakit.com](mailto:support@totakit.com) | Club member support | 24 hours |
| [@totakithq](https://x.com/totakithq) | Quick questions, updates | Best effort |

## Before Opening an Issue

1. Search [existing issues](https://github.com/totakit/totakit.github.io/issues) — someone may have already reported it
2. Check [Discussions](https://github.com/totakit/totakit.github.io/discussions) — it might already be answered
3. Try a different browser — some tools use Web APIs that vary by browser
4. Clear your browser cache — stale service workers can cause issues

## Club Members

If you're a [Club member](https://totakit.com/pricing), you get priority support:

- Email [support@totakit.com](mailto:support@totakit.com) with your account email
- Priority response within 24 hours
- Feature request prioritization
